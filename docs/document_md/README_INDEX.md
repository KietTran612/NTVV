# Farm Game Markdown Export Bundle

Bộ file Markdown được xuất từ toàn bộ các tài liệu `.docx` hiện có của project.

## File chỉ mục

- Tất cả file trong thư mục này là bản `.md` để dùng lại, tìm kiếm, commit Git hoặc đưa vào công cụ AI/dev.
- `farm_game_docs_all_in_one.md` là bản gộp toàn bộ nội dung.
- Danh sách dưới đây giữ nguyên cả các bản v1/v2/v3 để tiện đối chiếu lịch sử.
